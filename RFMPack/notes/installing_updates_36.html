Installing Updates

Normally, upon new minor release, two new files will be available for download:

•The first is the full version of the pack which contains all updates and changes up to, and including, the new release.

•The second is a, usually, smaller file for people who already downloaded the full pack and all they want is the latest files with the changes and/or additions.

If you download the update file, all you need is to copy, and overwrite when prompt, the files to the relevant paths.


The folder structure within each of the downloadable files is kept the same.


NOTE: In cases of a major version release (1.0, 2.0 etc.) it would be best to delete the old files before copying the new release. Then, copy the minor releases files to their designated folders (in some cases overwriting) by order of version release (2.1, 2.2 etc.).
